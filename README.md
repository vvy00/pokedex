# pokedex

Interactive Pokédex built with React that connects to the PokéAPI

# Features
* Live Search: Find any Pokémon by name or ID number.
* Randomizer: Discover new Pokémon with a "Random" button.
* Dynamic Data: Displays official artwork, types, height, and weight.
* Custom UI: A classic handheld Pokédex design with a custom background.
